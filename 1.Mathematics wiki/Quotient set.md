> [!quote] Definition: Quotient set
> Let $X$ be a set and $\sim$ be a equivalence relation on $X$, then we define $$ X/\sim := \{ [x] : x \in X\}$$
>

Where $[x]$ is the equivalence class of $x$ with respect to $\sim$.
